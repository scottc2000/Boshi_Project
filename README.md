Controls:
  Mario:
    - wasd movement
  Luigi:
    - ijkl movement
  Blocks/Obstacles
    - t, y cycle between blocks
  Items:
    - u, i cycle between items
  Enemy/NPC:
    - o, p cycle between enemies
  Other:
    - q quit
    - r reset to initial state
