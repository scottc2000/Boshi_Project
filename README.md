# Sprint 2
## Controls:
  ### Mario:
    - wasd movement
    -D1, D2, D3, D4 state changes
  ### Luigi:
    - ijkl movement
  ### Blocks/Obstacles
    - t, y cycle between blocks
  ### Items:
    - u, i cycle between items
  ### Enemy/NPC:
    - o, p cycle between enemies
  ### Other:
    - escape quit
    - D0 reset to initial state
